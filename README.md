Convertitore HTML

Questo è un semplice ma potente convertitore HTML progettato per funzionare in locale.  
Il progetto nasce da un’idea personale ed è stato sviluppato con il supporto di un modello di intelligenza artificiale.

Funzionalità
- Conversione rapida e immediata
- Interfaccia semplice e leggibile
- Compatibile con qualsiasi browser
- Utilizzabile anche da file locali

Motivi della creazione
Volevo realizzare uno strumento leggero e immediato, che potesse funzionare senza installare nulla.  
È uno dei miei primi progetti pubblici su GitHub.

Tecnologie utilizzate
- HTML5
- CSS3 
- JavaScript Vanilla

Note
Il codice è stato generato e ottimizzato con l’aiuto di un’IA, ma l’idea, la struttura e la pubblicazione sono completamente personali.
